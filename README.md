## Featured open source

### [React Native Device Risk Signals](https://github.com/AfanasievN/react-native-device-risk-signals)

An open-source React Native TurboModule for collecting raw device intelligence on Android and iOS.
It exposes root and jailbreak indicators, emulator and instrumentation traces, VPN/proxy state,
hardware, locale, application, and runtime signals—without assigning a client-side risk verdict.
